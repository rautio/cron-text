# Crontext

[![codecov](https://codecov.io/gh/rautio/crontext/graph/badge.svg?token=2D9ALEL4AD)](https://codecov.io/gh/rautio/crontext)

Parses natural text to a cron schedule format.

```js
parseCron('Every weekday at 9am');
// 0 9 * * 1-5
```
